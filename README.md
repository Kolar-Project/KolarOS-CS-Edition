# KolarOS Landscaper, codename CSharp Edition
KolarOS Landscaper is an operating system only for your needs.

## Requeriments

## Minimum:
RAM: 12MB.
CPU: a 75 MHz CPU.
HDD: 100MB

## Recommended:
RAM: 256MB or above.
CPU: a 200MHz CPU with MMX instructions or better.
HDD: 1GB or better.
